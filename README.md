🎮 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. This project allows users to play against the computer and keeps track of scores using local storage.

 🚀 Features

* Play Rock, Paper, Scissors against the computer
* Random computer move generation
* Score tracking (Wins, Losses, Ties)
* Data stored using **localStorage**
* Reset score functionality
* Clean and responsive UI

🛠️ Technologies Used

* HTML
* CSS
* JavaScript

📸 How It Works

1. Click on Rock, Paper, or Scissors
2. Computer randomly selects a move
3. Result is displayed instantly
4. Score updates automatically
5. Data is saved even after refresh
